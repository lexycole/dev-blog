Functions

Functional 프로그래밍에서 function

  1. Total
  • 모든 입력에 대해 대응하는 출력이 있다.
  1. Deterministic
  • 같은 입력값에 대해 같은 출력을 가진다.
  1. No Observable Side-Effect
  • 값을 계산하는 것 이외에 사이드 이펙트는 없다.
