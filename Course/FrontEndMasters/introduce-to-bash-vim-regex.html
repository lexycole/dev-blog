Using Shell

  • 어떤 쉘을 쓰는지 확인

    echo $SHELL
# /bin/bash

  • 디렉터리내 파일 리스트

    ls
ls /dir/path

  • 현재 디렉터리 확인

    pwd

  • 현재 디렉터리 이동

    cd /dir/path
cd - # cd 로 변경되기 바로 전 디렉터리로 이동

  • 별칭 지정

    alias ls='ls-l'

  • 별칭 확인

    alias

  • 특별한 디렉터리

    • ..: 상위(부모) 디렉터리
    • .: 현재 디렉터리
    • ~: 홈 디렉터리

  • 문자열 이어 붙히기

    cat test.txt test2.txt

  • 복사

    cp {source file} {target file}
cp -r {source directory} {target directory} # 디렉터리 내용을 포함해서 복사

mv

파일, 디렉터리의 이름을 바꾸거나 덮어쓰는데 사용

mv message.txt msg.txt
# message.txt -> msg.txt
# msg.txt 가 이미 있을 경우 덮어씀

mkdir

새로운 디렉터리를 만드는데 사용

mkdir foo # foo 디렉터리 생성
mkdir -p a/b/c/d # 디렉터리가 없으면 생성, 있으면 해당 디렉터리 사용

rm

파일, 디렉터리를 삭제하는데 사용

rm file # 파일 삭제
rm -r directory # 디렉터리 삭제

wc

writing, reading file

echo hello there > hello.txt # hello.txt 파일에 'hello there' 을 쓴다.
echo hello there 2 >> hello.txt # 'hello there 2' 를 hello.txt 뒤에 이어 붙힌다.

pipe |

ls -l | wc -l # 디렉터리에 있는 파일 갯수 출력
