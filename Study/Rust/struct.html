구조체

구조체(struct)는 튜플처럼 여러 데이터 타입으로 이루어질 수 있다. 하지만 튜플과 달리 각 데이터는 이름을 가지고 있고 데이터의 순서를 신경쓰지 않고 사용할 수 있다.

구조체 정의

struct User {
    active: bool,
    username: String,
    email: String,
    sign_in_count: u64,
}

구조체 인스턴스

정의한 구조체를 사용하기 위해서는 이를 인스턴스화 해야한다.

let user1 = User {
  email: String::from("someone@example.com"),
  username: String::from("someusername123"),
  active: true,
  sign_in_count: 1,
};
