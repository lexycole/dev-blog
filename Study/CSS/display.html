display

display: inline

  • 수평 방향으로 요소가 배치된다.
  • width, height 를 적용할 수 없다.
  • 수직 마진이 적용되지 않는다.

display: block

  • 수직 방향으로 요소가 배치된다.
  • width, height 를 적용할 수 있다.
  • 수직 마진이 서로 중첩된다.

display: inline-block

  • 수평 방향으로 요소가 배치된다.
  • width, height 를 적용할 수 있다.
  • 수직 마진이 중첩되지 않는다.

display: none

  • 화면, 스크린 리더에 표시되지 않는다

display: flow-root

display 속성을 바꾸는 속성

  • position: absolute | fixed
  • float: left | right

위 속성을 사용하게 되면 display: block 으로 강제 변경된다.

  • display: inline
  • display: block
  • display: inline-block
  • display: none
  • display: flow-root
